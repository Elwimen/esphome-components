# Elwimen's ESPHome Components

Custom ESPHome components and configurations.

## Components

- [fdx_reader](custom_components/fdx_reader) - FDX-B RFID tag reader
- [radiacode_ble](custom_components/radiacode_ble) - RadiaCode BLE radiation detector

## Configurations

- `fdx-tag-reader.yaml` - FDX-B RFID tag reader setup
- `radiacode.yaml` - RadiaCode radiation detector setup

## Usage

Each component is maintained in its own repository as a git submodule. Configuration files are provided as examples.
